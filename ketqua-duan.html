Ý tưởng thiết kế trang web báo tường

Chủ đề: Tri ân thầy cô & lưu giữ kỷ niệm đẹp của lớp 12A

  • Cấu trúc rõ ràng: Header (tiêu đề + slogan + ảnh nền) → Menu → Nội dung chính → Footer
  • Các phần nổi bật: Lời ngỏ, Bài viết hay, Ảnh & Đôi lời tâm sự, Góc bình luận
  • Responsive: Đẹp trên máy tính, điện thoại & máy tính bảng
  • Màu sắc: Xanh dương nhạt – trắng – điểm nhấn nhẹ nhàng , tạo cảm giác tươi trẻ, ấm áp

Mục tiêu: Trang web không chỉ đẹp mà còn chứa đựng tình cảm chân thành của tập thể lớp.

Các kỹ thuật HTML & CSS đã sử dụng

Kênh chữ

  • Thẻ <h1>–<h3>, <p>, danh sách <ul>
  • Font Google Fonts (Roboto & Dancing Script)

Kênh hình ảnh

  • <img> với border-radius, box-shadow, hover zoom
  • Ảnh nền header (background-image)

Tương tác & Responsive

  • Hover effect, transition, flexbox, media queries
  • Form bình luận demo, back-to-top button

Nhờ CSS linh hoạt, trang web trở nên bắt mắt và thân thiện trên mọi thiết bị.

Tự đánh giá sản phẩm & Bài học rút ra

Điểm mạnh

  • Giao diện đẹp, sinh động, đa kênh thông tin (chữ + hình )
  • Bố cục hợp lý, responsive tốt trên mobile & desktop
  • Nội dung ấm áp, phù hợp chủ đề tri ân & kỷ niệm lớp
  • Hoàn thiện trên GitHub Pages, dễ chia sẻ link

Điểm cần cải thiện & Bài học

  • Có thể thêm bình luận realtime (Utterances) ở phiên bản sau
  • Tối ưu tốc độ load ảnh/video lớn
  • Bài học: Làm việc nhóm hiệu quả, quản lý thời gian, hiểu sâu hơn về HTML/CSS, deploy web miễn phí
